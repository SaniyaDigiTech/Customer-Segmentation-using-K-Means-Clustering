# Customer Segmentation using K-Means Clustering

## Project Overview
This Machine Learning project applies the K-Means Clustering algorithm to segment customers based on their Age and Income. By grouping customers into similar categories, businesses can better understand customer behavior and improve marketing strategies.

## Features
- Data Cleaning and Preprocessing
- Feature Scaling using MinMaxScaler
- Customer Segmentation using K-Means Clustering
- Cluster Visualization with Matplotlib
- Centroid Identification and Analysis
- Unsupervised Machine Learning Implementation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Results
The model successfully identified three customer segments:

1. 🔴 Low Age, Low Income
2. 🔵 Medium Age, Medium Income
3. 🟡 High Age, High Income

Cluster centroids were calculated to represent the center of each customer group, helping visualize customer distribution patterns.
